# NeilCRM

A custom CRM for an infrastructure exchange — HubSpot replacement, tailored to surfacing the right **infrastructure operators** (data center, power, fiber, colo, etc.) and running **AI agents** that draft outreach, manage follow-ups, and surface "schedule a call" moments to a human.

Web-based. Independent auth. Modeled after [Monaco](https://www.monaco.com) (human-in-the-loop AI sales agents), narrowed to the infra vertical.
